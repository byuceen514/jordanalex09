    Software Design

    Purpose

    The pupose of this software is to provide a open access website for outdoor adventure seekers in Utah to be able to plan our hiking trips

    Structure

    Main Files

    There are 6 main files that are used to build this website. base.html, Help.html, home.html, map.html, techdoc.html, Video.html. Each of these files build a new page that contains different data the user can interact with. the only exception is base.html whis designs the format for which all the pages are based off of.

    Key Classes and Functions

    The main function used to calculate the elevations and plot the graph is based off a geoprocessing model from ArcGIS. The tool used is called Stack Profile
